YouTube Learning Library

Setup Guide for Mac

A step-by-step guide to get your personal video transcript library running

Setup time: ~30 minutes

What This Tool Does

This tool creates a personal, searchable library from YouTube videos. It downloads transcripts, uses AI to analyze and summarize content, and builds a beautiful website you can browse offline.

Before You Start

Make sure you have:

  • A Mac computer (any recent model)
  • At least 8GB of RAM
  • At least 10GB of free disk space
  • An internet connection
  • About 30 minutes of time

Setup Steps

1
Open Terminal

Terminal is the app where you'll type commands. Don't worry - we'll tell you exactly what to type!

To open Terminal:

  1. Press Command + Space on your keyboard (this opens Spotlight search)
  2. Type Terminal
  3. Press Enter

You'll see a window with a blinking cursor. This is where you'll paste the commands from this guide.

Keep Terminal open throughout this entire setup process.
2
Check Python Installation

Python is a programming language this tool is built with. Most Macs already have it installed.

Copy and paste this command into Terminal, then press Enter:

python3 --version
Python 3.11.6 (or any version 3.9 or higher)
What if Python is not installed?

If you see "command not found", you need to install Python:

  1. Go to python.org/downloads
  2. Click the yellow "Download Python" button
  3. Open the downloaded file and follow the installer
  4. Close Terminal and reopen it
  5. Try the command again
3
Install Required Packages

This installs the helper libraries the tool needs. Copy and paste this entire command:

pip3 install youtube-transcript-api requests jinja2 scrapetube flask whoosh

You'll see a lot of text scrolling. Wait until it finishes and you see your cursor again.

Successfully installed youtube-transcript-api-... requests-... jinja2-... (and more)
What do these packages do?
  • youtube-transcript-api - Downloads transcripts from YouTube videos
  • requests - Lets the tool talk to websites
  • jinja2 - Creates the HTML pages for your library
  • scrapetube - Finds all videos on a YouTube channel
  • flask - Powers the search feature
  • whoosh - Makes searching fast
4
Install Ollama (The AI Brain)

Ollama is a free app that runs AI on your Mac. It's what makes the tool smart - it analyzes videos and creates summaries.

Option A - Download from website (easiest):

  1. Go to ollama.ai
  2. Click "Download for macOS"
  3. Open the downloaded file and drag Ollama to your Applications folder
  4. Open Ollama from your Applications folder (you'll see a llama icon in your menu bar)

Option B - Using Homebrew (if you have it):

brew install ollama
5
Download the AI Model

Now we need to download the AI model that Ollama will use. This is about 4.7GB, so it may take 5-15 minutes depending on your internet speed.

ollama pull llama3.1:8b
pulling manifest
pulling 8eeb52dfb3bb... 100% |████████████████| 4.7 GB
success
This only needs to be done once. The model stays on your computer.
6
Start the Ollama Server

The AI needs to be running in the background while you use the tool.

ollama serve
This command keeps running and won't give you your cursor back. That's normal! It means the AI is ready and waiting.

Now open a NEW Terminal window:

  1. With Terminal in focus, press Command + N
  2. A new Terminal window will appear
  3. Use this new window for the remaining commands
If you downloaded Ollama from their website and it's running in your menu bar, you can skip this step - it starts automatically!
7
Navigate to the Tool Folder

In your new Terminal window, go to where you saved the YouTube Learning Library files:

cd ~/Desktop/youtube_transcript_to_md
If you saved the folder somewhere else, replace the path above with your actual location.
8
Test It Works!

Let's add your first video to make sure everything is working:

python3 youtube_transcript_to_md.py "https://www.youtube.com/watch?v=dQw4w9WgXcQ"

You'll see progress messages as the AI analyzes the video. This takes 1-3 minutes.

9
Generate Your Library Website

Now let's build the browsable website:

python3 library.py

Open site/index.html in a browser to view

To view your library, open the site folder and double-click index.html!

Common Commands

Here are the commands you'll use regularly:

Add a Single Video

python3 youtube_transcript_to_md.py "YOUTUBE_URL_HERE"

Import All Videos from a Channel

python3 channel_import.py "https://www.youtube.com/@ChannelName"

Import Multiple Videos from a List

Create a text file with YouTube URLs (one per line), then run:

python3 batch_import.py your_list.md

Rebuild the Website

Run this after adding new videos:

python3 library.py

Start the Search Server

This lets you search through all transcripts:

python3 search_server.py

Then visit http://localhost:5000 in your browser.

Troubleshooting

"command not found: python3"

Python isn't installed. Go to python.org/downloads and install it, then restart Terminal.

"command not found: pip3"

Try using python3 -m pip install instead of pip3 install.

"No transcript available" error

This video doesn't have captions/subtitles on YouTube. You can:

  • Try a different video
  • Use the manual import: python3 manual_import.py "URL" and paste the transcript yourself
"Ollama request failed" or AI analysis not working

Make sure Ollama is running:

  1. Check if the Ollama llama icon is in your menu bar (top of screen)
  2. If not, open Ollama from your Applications folder
  3. Or run ollama serve in a Terminal window
"Permission denied" error

Try adding sudo before the command and enter your Mac password when prompted:

sudo pip3 install youtube-transcript-api requests jinja2 scrapetube flask whoosh
Processing is very slow

This is normal! The AI runs locally on your Mac, which takes time. Each 3-minute section takes about 30-60 seconds to analyze. A 20-minute video might take 5-10 minutes total.

Newer Apple Silicon Macs (M1, M2, M3) are significantly faster than Intel Macs.

Glossary

Don't know what a term means? Here's a quick reference:

Terminal - An app on your Mac where you type text commands instead of clicking buttons. It's like having a conversation with your computer.

Python - A programming language. You don't need to learn it - just have it installed so the tool can run.

pip - A tool that installs Python add-ons (called "packages"). Think of it like an app store for Python tools.

Ollama - An app that runs AI models on your Mac. It's the "brain" that analyzes and summarizes your videos.

LLM (Large Language Model) - The type of AI that understands and generates text. ChatGPT is an LLM. Ollama runs one locally on your computer.

Transcript - The written text of everything said in a video, with timestamps.
